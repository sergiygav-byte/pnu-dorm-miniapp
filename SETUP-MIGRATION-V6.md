# Міграція v6 — переробка push, користувачі, санітарка

## Зміни

1. **Push** — один тумблер: увімкнено **усім** з `/start` або **нікому**.
2. **Користувачі** — зареєстровані + **зараз онлайн** (активність за 15 хв).
3. **Санітарка** — час (напр. 22:00), нагадування **зранку в день події**, статус виконано/ні, коментар для всіх.

## Кроки

1. Supabase → SQL Editor → **`supabase/migration_v6.sql`** → Run.
2. Vercel → **Redeploy** (`api/cron-duty-reminder.js`, `api/notify.js`).
3. GitHub → `git push` (`index.html`, `database.js`).

## Санітарка

- При додаванні: дата + **час** (напр. 1 травня, 22:00).
- **1 травня о ~09:00** — push «Сьогодні санітарка… о 22:00».
- Адмін: кнопка 📋 на записі → статус + коментар (видно всім).

## Push

**Інфо** → адмін → тумблер **«Сповіщення в Telegram»** — одразу зберігається при перемиканні.
