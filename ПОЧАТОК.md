## КРОК 3 — Vercel (бот + push) — 5 хв

1. https://vercel.com → проєкт **pnu-dorm-miniapp**
2. **Settings → Environment Variables:**

| Змінна | Звідки |
|--------|--------|
| `BOT_TOKEN` | @BotFather → ваш бот |
| `SUPABASE_URL` | Supabase → API |
| `SUPABASE_ANON_KEY` | Supabase → anon key |
| `NOTIFY_SECRET` | придумайте довгий пароль (= у `config.js` NOTIFY_SECRET) |
| `WEBAPP_URL` | `https://sergiygav-byte.github.io/pnu-dorm-miniapp/index.html` |

3. **Deployments → Redeploy**

4. Webhook для Telegram (один раз у браузері):

```
https://api.telegram.org/botВАШ_BOT_TOKEN/setWebhook?url=https://pnu-dorm-miniapp.vercel.app/api/webhook
```

У відповіді: `"ok":true`.

Деталі: `BOT-START.md`, push: `BOT-PUSH.md`.
