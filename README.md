# Гуртожиток №1 ПНУ — Telegram Mini App

Додаток для бюджету, подій, чергувань, скарг і контактів гуртожитку.

**Покрокова інструкція:** відкрий файл [SETUP.md](./SETUP.md) і роби тільки один крок за раз.

## Стек

| Сервіс | Навіщо |
|--------|--------|
| **GitHub** | Зберігання коду |
| **Supabase** | База даних + сховище фото |
| **Render** | Публічний сайт для Telegram Mini App |
| **Telegram** | Bot + Mini App |

## Локально відкрити сайт

```powershell
cd C:\Users\User\Projects\pnu-dorm-miniapp
npx --yes serve .
```

Потім у браузері: http://localhost:3000
